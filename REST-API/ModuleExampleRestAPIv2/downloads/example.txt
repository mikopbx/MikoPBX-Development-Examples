MikoPBX Module Example - REST API v2
=====================================

This is a demonstration file for testing file download functionality.

Download modes:
- mode=view: Returns file content as JSON
- mode=download: Returns URL for browser download

API Endpoints:

GET Operations:
  GET /pbxcore/api/module-example-rest-api-v2/config
  GET /pbxcore/api/module-example-rest-api-v2/users
  GET /pbxcore/api/module-example-rest-api-v2/download?filename=example.txt
  GET /pbxcore/api/module-example-rest-api-v2/download?filename=example.txt&mode=download

POST Operations:
  POST /pbxcore/api/module-example-rest-api-v2/create
  POST /pbxcore/api/module-example-rest-api-v2/update
  POST /pbxcore/api/module-example-rest-api-v2/delete
